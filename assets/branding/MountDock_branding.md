# MountDock Branding

## KR

### 브랜드 핵심
MountDock은 rclone 마운트를 쉽게 다루게 해주는 Windows용 데스크톱 유틸리티입니다.
브랜드 인상은 화려함보다 **정돈됨, 실용성, 안정감**에 가깝게 유지합니다.

### 키워드
- Docked
- Mounted
- Structured
- Practical
- Windows-native
- Calm blue

### 아이콘 방향
1. **Drive + Dock**
   - 드라이브 실루엣 아래에 도킹 바/받침을 둔 형태
2. **Mounted Volume**
   - 볼륨이 베이스 위에 놓인 듯한 간단한 형태
3. **MD Monogram**
   - `M`과 `D`를 단순한 선형 아이콘으로 결합

### 권장 방향
가장 추천하는 쪽은 **Drive + Dock** 입니다.
이유:
- 기능이 가장 바로 읽힘
- Windows 유틸리티 느낌이 남
- 너무 개발자 도구 같지도 않음

## EN

### Brand Core
MountDock is a Windows desktop utility that makes rclone mounts easier to manage.
Its visual identity should feel **structured, practical, and dependable**, not flashy.

### Keywords
- Docked
- Mounted
- Structured
- Practical
- Windows-native
- Calm blue

### Icon Directions
1. **Drive + Dock**
   - drive silhouette resting on a docking base
2. **Mounted Volume**
   - simplified volume shape placed on a base
3. **MD Monogram**
   - minimal merged `M` and `D` letterform

### Recommended Direction
The strongest option is **Drive + Dock**.
Why:
- the function reads immediately
- it fits a Windows utility well
- it does not feel overly developer-centric
