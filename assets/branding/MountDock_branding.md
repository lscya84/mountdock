# MountDock Branding

## KR

### 브랜드 핵심
MountDock은 rclone 마운트를 쉽게 다루게 해주는 Windows용 데스크톱 유틸리티입니다.
브랜드 인상은 화려함보다 **정돈됨, 실용성, 안정감**에 가깝게 유지합니다.

### 키워드
- Docked
- Mounted
- Structured
- Practical
- Windows-native
- Calm blue

### 아이콘 방향
1. **Duck + Dock**
   - `dock`의 언어유희로 `duck` 실루엣을 쓰되, 하단에 도킹 베이스를 두어 제품 정체성을 유지
2. **Drive + Dock**
   - 드라이브 실루엣 아래에 도킹 바/받침을 둔 형태
3. **MD Monogram**
   - `M`과 `D`를 단순한 선형 아이콘으로 결합

### 권장 방향
현재 가장 개성 있는 쪽은 **Duck + Dock** 입니다.
이유:
- MountDock 이름을 기억하게 만드는 요소가 있음
- 너무 가볍지 않게 만들면 유틸 앱에도 잘 어울림
- GitHub 배포 앱으로서 캐릭터성이 생김

## EN

### Brand Core
MountDock is a Windows desktop utility that makes rclone mounts easier to manage.
Its visual identity should feel **structured, practical, and dependable**, not flashy.

### Keywords
- Docked
- Mounted
- Structured
- Practical
- Windows-native
- Calm blue

### Icon Directions
1. **Duck + Dock**
   - a duck silhouette built around the dock wordplay, with a docking base to preserve product meaning
2. **Drive + Dock**
   - drive silhouette resting on a docking base
3. **MD Monogram**
   - minimal merged `M` and `D` letterform

### Recommended Direction
The most distinctive direction now is **Duck + Dock**.
Why:
- it makes the MountDock name more memorable
- it can still feel practical if kept geometric and clean
- it gives the GitHub-distributed app stronger personality
